 To execute qHiPSTER programs from the test/ directory in the 
 ccefw-desktop.jf.intel.com (Ubuntu) environment, you need to setup the 
 LD_LIBRARY_PATH variable to point to the Intel64 Compiler libraries in
 /opt/intel/  .

 # For a Bash environment.
 $ export LD_LIBRARY_PATH = $LD_LIBRARY_PATH:/opt/intel/mkl/lib/intel64
 $ export LD_LIBRARY_PATH = $LD_LIBRARY_PATH:/opt/intel/composer_xe_2015.2.164/compiler/lib/intel64
